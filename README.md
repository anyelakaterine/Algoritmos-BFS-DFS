# Implementación de BFS y DFS en Python

Este proyecto implementa dos algoritmos de búsqueda en grafos: BFS (Breadth First Search) y DFS (Depth First Search), utilizando Python y ejecutados en Google Colab.

## Objetivo

Programar y comparar los algoritmos BFS y DFS utilizando un grafo pequeño, observando el camino encontrado y una estimación del uso de memoria.

## Algoritmos implementados

### BFS (Breadth First Search)
BFS explora el grafo por niveles utilizando una estructura de datos tipo cola (FIFO: First In First Out). Este algoritmo permite encontrar el camino más corto en grafos no ponderados.

### DFS (Depth First Search)
DFS explora el grafo en profundidad utilizando una pila (LIFO: Last In First Out). El algoritmo sigue un camino hasta el final antes de retroceder y explorar otras rutas posibles.

## Funcionamiento del programa

1. Se define un grafo pequeño representado mediante un diccionario en Python.
2. Se ejecuta el algoritmo BFS para encontrar una ruta entre un nodo inicial y uno objetivo.
3. Se ejecuta el algoritmo DFS para encontrar otra posible ruta.
4. El programa muestra:
   - El camino encontrado por cada algoritmo.
   - Una estimación del uso de memoria durante la ejecución.
